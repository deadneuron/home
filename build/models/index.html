Cataloging Model Architectures

A lot of interesting neural networks have been introduced over the years, but there is no place that organizes and discusses some of these fascinating models. This is a new take on the model zoo.

  • LeNet (1989)
  • Hopfield Networks (1982)
  • Long Short-Term Memory (1989)
  • Boltzmann Machine (1986)
  • Echo State Network (2004)
  • AlexNet (2012)
  • Inception (2014)
  • VGG (2014)
  • ResNet (2015)
  • ResNext (2016)
  • DenseNet (2016)
  • SqueezeNet (2016)
  • MobileNet (2017)
  • Transformer (2017)
  • MLP Mixer (2021)
  • EfficientNet (2019)
  • ConvNext (2022)
